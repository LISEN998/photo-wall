照片墙播放系统 - 使用说明
项目简介
照片墙播放系统是一个基于Web的幻灯片播放系统，支持全屏自动轮播照片和背景音乐播放。适合家庭照片墙展示、店铺产品展示、会议演示等场景。

系统特性
✅ 自动扫描照片和音乐文件

✅ 多种炫酷切换效果（淡入淡出、滑动、缩放、旋转等）

✅ 背景音乐播放

✅ 全屏/窗口模式切换

✅ 响应式设计，支持各种屏幕尺寸

✅ 键盘快捷键操作

✅ 自动隐藏控制条（全屏时）

快速开始
方法1：一键启动（推荐）
双击 start.bat 文件

系统将自动扫描文件并启动服务器

打开浏览器访问：http://localhost:8080

方法2：手动启动
将照片放入 assets/photos/ 文件夹

将音乐放入 assets/music/ 文件夹

双击 index.html 直接打开（仅照片模式）

或运行 python -m http.server 8080 启动完整功能

高级功能
自动扫描
系统会自动扫描指定文件夹内的照片和音乐文件，按文件名排序。

预加载功能
系统会预加载下一张照片，确保切换流畅。

图片缓存
已加载的图片会缓存，提高重复播放的性能。

响应式设计
自适应各种屏幕尺寸，支持4K大屏到手机屏幕。

全屏优化
全屏模式下控制条会自动隐藏，鼠标移动时显示。

常见问题
1. 看不到照片？
检查照片是否放在 assets/photos/ 文件夹

检查照片格式是否受支持

运行 scan.bat 重新扫描文件

2. 音乐播放卡顿？
确保音乐文件是标准MP3格式（建议128kbps）

检查音乐文件是否损坏

尝试转换为Ogg Vorbis格式

3. 切换效果不流畅？
检查照片尺寸是否过大（建议压缩到1920×1080）

降低切换速度设置

关闭其他占用资源的程序

4. 无法全屏？
确保浏览器允许全屏

尝试按F11键

检查是否有浏览器扩展阻止全屏

5. 控制条不自动隐藏？
检查是否在全屏模式

鼠标不要停留在控制条区域

等待3秒自动隐藏
